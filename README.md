
# Dark Theme Production Tracker

Live Demo: https://cod-production-tracker.netlify.app/

A production tracker dashboard built with React, Vite, and Tailwind CSS.

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui

## Features
- Dark theme dashboard UI
- Component-based architecture
- Responsive layout
- Deployed on Netlify

## Getting Started

npm install  
npm run dev  

## Deployment

Built with Vite and deployed using Netlify.
